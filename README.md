# 🚀 AI Typing Coach

A modern typing speed game with real-time analytics, game modes, and AI-style personalized feedback.

🌐 **Live Demo:**
https://yusuferoglu52.github.io/ai-typing-coach/

---

## 🎮 Features

### 🧠 Smart Typing Analysis

* Real-time **WPM (Words Per Minute)** tracking
* **Accuracy calculation**
* **Mistake detection**
* Progress tracking

### 🤖 AI Coach

* Personalized feedback after each session
* Structured insights:

  * **Strength**
  * **Weakness**
  * **Next Drill**
* Based on typing performance and behavior

### 🎯 Game Modes

* ⏱️ **Time Mode** – Type as much as you can within a fixed time
* ❤️ **Survival Mode** – 3 lives, each mistake costs one
* 🔥 **Streak Mode** – Build the longest correct typing streak
* 🧩 **Memory Mode** – Memorize text, write before it vanishes

### 📊 Performance Analytics

* Top **mistyped letters tracking**
* Foundation for **keyboard heatmap visualization**
* Data-driven feedback system

### 🎚️ Difficulty Levels

* Easy / Medium / Hard / Extreme
* Varying sentence length and complexity

## ⚙️ How It Works

* The app runs entirely in the browser (no backend)
* Typing input is analyzed in real time
* Game logic updates:

  * speed
  * accuracy
  * mistakes
  * streaks / lives
* AI Coach generates rule-based feedback from user stats
